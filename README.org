* Load Balancing Neo4J Bolt with Traefik

** Run

#+BEGIN_SRC shell
docker-compose up -d
#+END_SRC

** Traefik Dashboard

[[http://localhost:8080][Dashboard]]

** Get Traefik ip address
#+BEGIN_SRC shell
docker inspect traefik | grep '"IPAddress"'
#+END_SRC

** Port testing
#+BEGIN_SRC shell
docker exec -ti cypher-shell bash
root@c41f646a0de7:/var/lib/neo4j# apt-get update -qqy && apt-get install netcat -qqy # install netcat for port debugging
root@c41f646a0de7:/var/lib/neo4j# nc -zv <replace-with-traefik-ip-address> 7687
Connection to 172.22.0.2 7687 port [tcp/*] succeeded!
#+END_SRC

** Test bolt Connection via Traefik
#+BEGIN_SRC shell
docker exec -ti cypher-shell bash
cypher-shell -a bolt://172.22.0.2:7687 -u neo4j -p password
Connected to Neo4j using Bolt protocol version 5.7 at bolt://172.22.0.2:7687 as user neo4j.
#+END_SRC
